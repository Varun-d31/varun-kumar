HTML, CSS & Java Script

HTML

HTML stands for Hyper Text Markup Language. It is called as presentation language. It is also called as formatting language. It contains tags, attributes & entities.

CSS

CSS stands for Cascading Style Sheets. It is a style sheet language & it is used to style web pages. It contains properties & selectors. There are many properties & selectors.

Java Script

Java Script was initially object based & now it is an object oriented. It is used to add dynamic behaviour to a web page. It is also used to validate html forms.
